## 📋 Sheet 3: TESTIMONI (untuk Review Pelanggan)

Buat sheet dengan nama **"testimoni"** dengan struktur kolom:

| A | B | C |
|---|---|---|
| Nama Pelanggan | Rating (1-5) | Testimoni |
| Budi Santoso | 5 | Produk sangat bagus! |
| Siti Rahayu | 5 | Pengiriman cepat! |

---

## 🚀 Cara Menggunakan:

1. Buka **Google Sheets** baru
2. Buat **3 sheet** dengan nama: `banner`, `produk`, `testimoni`
3. Isi data sesuai template di atas
4. Klik **File → Share → Publish to web**
5. Pilih **Entire Document** dan format **CSV**
6. Klik **Publish**
7. Copy link yang diberikan dan发给 saya

---

## 📝 Catatan Penting:

- Kolom pertama (A) WAJIB ada judul
- Pastikan gambar menggunakan link BUKAN path lokal
- Link gambar harus bisa diakses publik (tidak private)
- Link Shopee harus lengkap (https://shopee.co.id/...)
